# _모음-화학-화학의언어 — 열람용 전체 자료 사본 (A to Z)

화학 「화학의 언어」 수행평가가 평가 프로토콜 4단계(준비→실시→결과→회고)를 거치며 만든
**모든 산출물의 열람용 사본**이다. 파일명 앞 번호가 프로토콜 순서다.

- **여기서 시작**: `00_목차.html`을 브라우저로 열면 전체 자료를 한 파일에서 훑을 수 있다 (파일별 링크 포함).
- **원본(진실의 원천)**: `assessments/화학-화학의언어/` — **수정은 반드시 원본에서** 하고, 필요하면 이 사본을 다시 만든다.
- 폴더명이 `_`로 시작하므로 lint 검사 대상이 아니다 (원본만 검사).
- 모든 문서는 **초안**이며 교사 검토·확정을 전제한다. 학생은 번호로만 표기(비식별).
